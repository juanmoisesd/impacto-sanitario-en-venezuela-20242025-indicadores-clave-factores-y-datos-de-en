## Overview
Esta base de datos recopila indicadores clave de salud, datos de brotes de enfermedades, tasas de vacunación, estado del sistema de salud, seguridad alimentaria y los principales factores que influyen en la salud pública en Venezuela durante 2024–2025. Datos obtenidos de Perplexity.ai, con cita de OMS/OPS, HRW, UNICEF, OCHA, EU ECHO, Ballard Brief (BYU), Disaster Philanthropy, World Vision, The Gl
